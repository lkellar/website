FTView has been shutdown

Unfortunately, the switch to Bungie Names for Destiny 2 broke FTView. I am no longer able to maintain the application so I decided to shut it down. If you want to explore the source code or fix it and run it yourself, the code is available on Github

However, there are some excellent alternative applications if you're looking for quick sites to lookup guardians. Here's a list of my favorites

  • trials.report
  • guardian.report
  • crucible.report
  • raid.report

Additionally, Braytech and Destiny Item Manager are also great webapps for managing your personal inventory and statistics.
